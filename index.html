Agent-Ready Evidence Standard

Public, attributable signals
B2B relevance filter

Include enterprise buyers, finance leaders, platforms, merchants, banks, acquirers and commercial operators. Exclude purely consumer activity unless it affects B2B positioning.

Public signal sources

Campaign pages, product pages, press releases, event pages, thought leadership, webinars, customer stories, executive posts and partner announcements.

Competitiveness model

Score each segment on four parent categories, expand each parent into sub-metrics, then calculate the weighted 0-100 competitiveness score from those underlying inputs.

Memory files

Keep durable context in competitor-baseline.md, weekly-change-log.md and whitespace-topics.md so changes are visible over time.

Real Agent Blueprint

How this page becomes live
A scheduled research worker gathers public evidence, applies the B2B relevance filter, and writes structured JSON for the dashboard.
The model produces competitor movement, whitespace topics, ranked bets and buyer-facing creative briefs from the instruction set.
GitHub Pages can remain the front-end, but API keys and web research must run on a backend such as Vercel, Netlify Functions or Cloudflare Workers.
